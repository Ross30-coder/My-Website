How to Install an SSD Hard drive and load Windows

Tools and Accessories Required

Tool/Accessories Purpose Required?
Phillips Head Screwdriver To Secure the SSD to the case Yes
SATA Cable Connect the SSD to motherboard Yes
Anti-static Wrist Strap Protect components from static electricity damage No but recommended
USB Drive 8GB+ Windows installation media Yes

Step 1:Install the SSD

  1. Make sure your PC is powered of and unpluged.
  2. Open the side panel of your case
  3. Locate the drive bay
  4. Secure the SSD in place using screws
  5. Connect the SATA and power cables
  6. Close the case and plug back in the PC


Step 2:Create a Windows Installation USB

  1. On another computer, go to Microsoft's Offical website.
  2. Download the Media Creation Tool.
  3. Run the software tool and create a bootable USB drive with at least 8GB of storage.


Step 3:Install Windows

  1. Insert the USB into your system and turn it on.
  2. Enter the BIOS by pressing Del or F2 during startup.
  3. Set the USB drive as the primary boot device.
  4. Save and exit BIOS. The PC should boot from the USB.
  5. Follow the on-screen instructions to install Windows.
  6. When asked where to install Windows, select your new SSD.
